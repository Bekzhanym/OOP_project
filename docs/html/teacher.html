kbtu_oop_project.domain.features.user.Teacher

Роль: сотрудник и исследователь, ведущий курсы.

  • Связь с курсами (taughtCourses), частично восстанавливается из графа Course→Teacher при загрузке.
  • putMark — выставление оценки студенту по курсу с проверками домена.
  • manageCourse, exportGradeReport, writeRecommendation — из UML; в консоли — управление курсом и отчёт по оценкам.
  • ResearchPaper и ResearchProject как у исследователя.
  • Подтип Professor задаёт учёную степень по умолчанию.
