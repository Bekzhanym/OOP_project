kbtu_oop_project.console.features.manager.ManagerConsole

Роль: меню менеджера в текстовом интерфейсе.

  • Отчёты по оценкам и рейтинг студентов по GPA transcript.
  • Назначение преподавателя на курс.
  • Очередь заявок на курсы: утверждение (a номер) или отклонение (r номер) через БД.
  • Просмотр внутренней почты с флагом подписи декана.
  • Список преподавателей по алфавиту и агрегированный средний рейтинг по всем студентам.
  • Добавление курса в каталог — с CourseType, major, годом обучения и опциональным Lesson.
  • Новости университета — просмотр и добавление (интеграция с точкой расширения Manager.manageNews()).
